## About the Project

Your Spicemen is a premium e-commerce platform specializing in authentic Indian masalas and carefully curated tea blends. Built with Next.js, TypeScript, and Tailwind CSS, this website offers a modern and intuitive shopping experience for spice enthusiasts and tea lovers.

Our collection features:
- **Premium Masalas**: Garam Masala, Kashmiri Lal Mirch, Organic Turmeric, Pure Asafoetida (Hing), and more
- **Authentic Teas**: Heritage blends, Herbal infusions, Imperial flush teas, House chai varieties
- **Traditional Quality**: Handpicked spices and tea leaves sourced directly from Indian farms
- **Modern Shopping**: Clean, responsive design with seamless user experience

👉 **Live Demo**: [Your Spicemen E-commerce](#)

## Features

✨ **Product Showcase**: Beautiful product galleries with detailed descriptions  
🛒 **Shopping Cart**: Add to cart functionality with quantity management  
📱 **Responsive Design**: Optimized for all devices  
🎨 **Modern UI**: Clean, minimalist design focusing on products  
🔍 **Category Filtering**: Browse by Masala or Tea categories  
🏪 **Easy Navigation**: Intuitive menu with Home, About Us, Products, and Contact

## Built With

- **Next.js 13** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first CSS framework
- **React Context** - State management for cart and UI toggles

## Getting Started

```bash
# Clone the repository
git clone https://github.com/your-username/your-spicemen.git

# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

## Project Structure

```
app/
├── components/          # Reusable UI components
├── context/            # React Context providers
├── data/              # Product data and categories
├── categories/        # Category pages
├── product/          # Individual product pages
└── globals.css       # Global styles
```
